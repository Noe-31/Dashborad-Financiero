# Dashboard Financiero

Proyecto React + TailwindCSS para un panel financiero moderno y funcional.

## Características
- Sidebar moderna
- Estadísticas financieras
- Gráficas dinámicas
- Responsive design
- Dark mode
- Tabs
- Modales
- Notificaciones
- Loader
- Buscador
- Filtros
- Exportar PDF
- Animaciones suaves
- Datos guardados en LocalStorage

## Módulos
- Ingresos
- Gastos
- Categorías
- Metas financieras
- Historial de movimientos
- Reportes y estadísticas

## Uso
```bash
npm install
npm run dev
```

El proyecto se abrirá en `http://localhost:5173`.
